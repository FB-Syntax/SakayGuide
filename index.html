SakayGuide Journey

From MVP to full-featured commuter companion — here's our planned path.

1
✅ v1.0 — Current

MVP: Live Dashboard, Gas Finder, Fare Calc & Route Map

Real-time jeepney & tricycle fares, GasWatchPH fuel prices with nationwide averages, gas station finder with search & brand filter, and interactive Intramuros terminal map.

  • Live gas prices from GasWatchPH API with nationwide averages
  • Gas Station Finder — search by name/brand/area with brand filter
  • Nearby station auto-detect via GPS + IP geolocation
  • Interactive gas station map with brand-colored markers & popups
  • PH Fuel Price dashboard card with animated counters
  • LTFRB-based jeepney fare (P14 first 4km + P2/km)
  • Manila LGU tricycle fare (P16 first km + P10/km)
  • Interactive Leaflet map with terminal markers
  • Live route drawing: Mapúa → destination by distance
  • Mobile-first white/blue responsive design

Route Finder & Fare Matrix Export

Add detailed jeepney/tricycle route listings for Intramuros and nearby barangays with exportable fare tables.

  • Route listings with terminal photos & hours
  • Fare matrix PDF / image export
  • Search by origin & destination
  • More Intramuros routes & landmarks
